A Modern Web Component Delivery Framework

Using Stenciljs and NPM to build and deliver web components to any team, any framework, at lightning speed ⚡

What

• Build Native Web Components that are supported by all browsers (ie & edge with polyfills) • Publish your components to private Accenture scoped Org

Why

Framework Agnostic Angular React Vue Vanilla HTML/JS • Improves developer onboarding by 10x because they can immediately deliver feature rich components without the need for client/project specific environment setup • This in turn improves project resource management by allowing PM’s and architects to focus on component specifications and allows developers to focus on implementation • Using NPM is the only way to make components easily indexed and consumed in projects with just one command (+ it's a tool all developers are already innately familiar with)

How

Stenciljs only requires some jsx and modern ES6 Javascript knowledge. With its gradual learning curve, developers can onramp smoothly and seamlessly

How (Continued)

The integratation team can simply include the web component into any framework with just two lines of code after it is published to npm

Summary and Demos
